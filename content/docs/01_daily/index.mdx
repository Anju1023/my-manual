# 📅 毎日の業務 (Daily Routine)

毎日行う事務作業およびシステム操作の手順です。
以下のタイムラインに沿って業務を遂行してください。

<Callout title="使用するファイル" icon={<FileText />}>
	以下のファイルはデスクトップの **「よく使う」** フォルダに入っています。 
    
    * **日替わり表:** `CB商品データ.xlsm` 
    * **納品記録:** `T納品データ.xlsm`

</Callout>

## 🕒 作業タイムライン

| 期限        | 作業内容                                                                        |
| :---------- | :------------------------------------------------------------------------------ |
| **~ 10:00** | **① 日替わり表の印刷**<br/>パートスタッフが出勤する前までに完了させてください。 |
| **当日中**  | **② 納品記録の保存**<br/>手が空いたタイミングで実施してください。               |

<br />

## ⚠️ 注意事項

- **マクロ実行中の操作禁止**

  処理中（画面が動いている間）は、キーボードやマウスに触れないでください。処理が中断したり、Excel がフリーズする原因となります。

- **エラー発生時の対応**

  もしエラー画面が表示された場合や、正常に動作しない場合は、そのまま操作せずにシステム担当（結城）へ連絡してください。
