# 🎛️ 操作シート・ボタン機能一覧

「操作」シートに配置されているボタンと、実行される VBA マクロの対応表です。
ボタンの配置が変わったり、機能を追加する際の参考にしてください。

## 🚀 一括出力系 (メイン機能)

日常業務で主に使用する、複数の処理をまとめて行うボタンです。

| ボタン名                       | 実行マクロ                   | 機能概要                                                                                             |
| :----------------------------- | :--------------------------- | :--------------------------------------------------------------------------------------------------- |
| **全部まとめて出力**           | `全部まとめて出力`           | 以下の「製造数」「ラベル」「生地出し」「発注」をすべて一括で実行します。土曜日の更新後に使用します。 |
| **合計まとめて出力**           | `合計まとめて出力`           | 「CB 製造数」「クリエイト製造数」「製造数(合計)」の 3 シートを再集計します。                         |
| **まとめてラベル出し出力**     | `まとめてラベル出し出力`     | ローゼン用とクリエイト用の「消費期限チェック表」を一括出力します。                                   |
| **生地出し並べ表まとめて出力** | `生地出し並べ表まとめて出力` | 「冷凍生地」「スクラッチ」「並べる表」の 3 つを一括出力します。                                      |

## 🏭 個別出力系 (メンテナンス用)

特定の帳票だけを再出力したい場合に使用します。

### 製造数・集計

| ボタン名                 | 実行マクロ                 | 備考                        |
| :----------------------- | :------------------------- | :-------------------------- |
| **CB 製造数出力**        | `CB製造数合計出力`         | ローゼン店舗分の集計        |
| **クリエイト製造数出力** | `クリエイト製造数合計出力` | クリエイト納品分の集計      |
| **製造数出力**           | `製造数合計出力`           | 上記 2 つを合算した総製造数 |

### 工程表・ラベル

| ボタン名                     | 実行マクロ                 | 備考                                 |
| :--------------------------- | :------------------------- | :----------------------------------- |
| **冷凍生地出し表出力**       | `冷凍生地出し表出力`       | 前日解凍が必要な生地のリスト         |
| **スクラッチ生地出し表出力** | `スクラッチ生地出し表出力` | 当日仕込み（スクラッチ）生地のリスト |
| **並べる表出力**             | `並べる表出力`             | 天板への配置数（〇枚 × 〇個）を計算  |
| **ラベル出し表出力**         | `消費期限チェック表出力`   | ラベラー登録用の個数表               |

### 発注・納品

| ボタン名                   | 実行マクロ                           | 備考                                                   |
| :------------------------- | :----------------------------------- | :----------------------------------------------------- |
| **発注表出力**             | `発注表作成` ➔ `発注表_仕入先別出力` | 原材料の所要量を計算し、仕入先ごとにシート分けして出力 |
| **商品別数量表(ローゼン)** | `商品別シート出力`                   | 店舗ごとの仕分けリスト（別ブック作成）                 |
| **商品別数量表(クリエ)**   | `クリエイト商品別シート出力`         | クリエイト用の仕分けリスト（別ブック作成）             |

## ⚙️ システム制御系

| ボタン名     | 実行マクロ      | 機能概要                                                                             |
| :----------- | :-------------- | :----------------------------------------------------------------------------------- |
| **更新**     | `更新`          | **【重要】** 土曜日に実行。翌週分のデータを今週分にコピーし、日付を 1 週間進めます。 |
| **新規入力** | `frmInput.Show` | 新商品登録用のユーザーフォームを起動します。                                         |

<Callout type="info" title="開発者メモ" icon={<MousePointerClick />}>
	**「更新」ボタンの挙動について** 「更新」マクロは、各シートの
	`J列〜P列`（翌週エリア）を
	`C列`（今週エリア）に値貼り付けする処理を行っています。
	そのため、**実行前にバックアップを取らないと、今週分のデータが消えて復元できなくなります。**
</Callout>
