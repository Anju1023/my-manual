## 📊 システム構成とデータ連携

本システムのファイル構成と、データの流れ（連携）です。
特に **`CB商品データ` と `T納品データ` は自動連携していない** 点に注意してください。

<Callout type="info" title="データの流れ (Data Flow)" icon={<Workflow />}>
    * **CB商品データ.xlsm** ➔ (マクロ) ➔ **🖨️ 日替わり表**
    * **T納品データ.xlsm** ➔ (マクロ) ➔ **📄 納品記録PDF**
    * **シフト表.xlsx** ➔ (手動) ➔ **📱 シフト画像**

    ※ **CBデータ** から **T納品データ** への連携（ID・商品名）は **手動転記** が必要です。

</Callout>

## 📝 重要ファイル・パス一覧

システムが参照・出力する主要なファイルの場所です。

| ファイル/フォルダ      | パス (ショートカット先)        | 用途                 |
| :--------------------- | :----------------------------- | :------------------- |
| **CB 商品データ.xlsm** | `Desktop\よく使う\商品データ\` | 製造指示、新商品登録 |
| **T 納品データ.xlsm**  | `Desktop\よく使う\伝票作成\`   | 納品書発行、売上集計 |

## 🚨 緊急時対応フロー

システムエラー（マクロが止まる、画面が固まる等）が発生した場合の対応手順です。

1.  **画面の保存（証拠確保）**
    - エラーメッセージが出ている画面をスマホ等で撮影してください。
    - （「終了」「デバッグ」等のボタンはまだ押さないでください）
2.  **強制終了**
    - エラー画面の **「終了」** ボタンを押して閉じます。
    - もし固まって動かない場合は、タスクマネージャー等から Excel を強制終了してください。
3.  **管理者への連絡**
    - 撮影した画像と共に、以下の担当者へ状況を報告してください。

<Callout type="error" title="システム担当連絡先" icon={<Phone />}>
	* **担当者:** 結城 
    * **連絡手段:** LINE または 電話

</Callout>
