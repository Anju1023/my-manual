# 🗓️ 毎週の業務 (Weekly Routine)

週に数回、決まった曜日に行う業務のマニュアルです。
曜日ごとのタスクを確認し、漏れがないように実施してください。

## 📅 週間スケジュール

| 曜日       | 業務内容           | 備考                 |
| :--------- | :----------------- | :------------------- |
| **火曜日** | **消耗品の発注**   | 18:00 までに送信     |
| **木曜日** | **消耗品の発注**   | 18:00 までに送信     |
| **土曜日** | **データ更新作業** | 2 週間ローリング処理 |
