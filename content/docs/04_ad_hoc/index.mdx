# 🛠️ 随時行う作業 (Maintenance)

新商品の登録や価格変更、数量調整など、必要に応じて行うメンテナンス業務のマニュアルです。
これらの作業は **システムの中身（マスタデータ）を書き換える** ため、慎重に行う必要があります。

## 🛡️ 【最重要】バックアップのルール

<Callout type="error" title="変更前のバックアップ（必須）" icon={<AlertTriangle />}>
	**変更を加える前に、必ずファイルのコピー（バックアップ）を作成してください。**

    操作ミスによる数式の削除や、ファイルの破損が発生した場合、バックアップがないと復旧できません。
    作業終了後は、以下の手順に従って別名で保存してください。

</Callout>

### バックアップの手順

<Steps>
<Step>

### ファイルをコピーする

以下のフォルダにある `CB商品データ.xlsm` を選択し、コピーします。

- **場所:** `デスクトップ` > `よく使う` > `商品データ`

<Callout type="info" title="ショートカットキー" icon={<Keyboard />}>
	ファイルを選択して **`Ctrl` + `C`** キーを押すとコピーできます。
</Callout>

</Step>
<Step>

### バックアップ先を開く

同じ `よく使う` フォルダの中にある **`BackUp`** フォルダを開き、ファイルを貼り付けます。

- **保存先:** `デスクトップ` > `よく使う` > `BackUp`

<Callout type="info" title="ショートカットキー" icon={<Keyboard />}>
	フォルダを開いて **`Ctrl` + `V`** キーを押すと貼り付けできます。
</Callout>

</Step>
<Step>

### ファイル名の変更

貼り付けたファイルの名前を変更します。
「いつ」「何をしたか」が明確になるよう、以下のルールで命名してください。

![バックアップフォルダの例](https://placehold.co/600x400?text=Backup+Folder+Example)

- **命名規則:** `元のファイル名` + `日付` + `変更内容`
- **例:** `CB商品データ2025-12-16〇〇店変更.xlsm`

<Callout type="info" title="ショートカットキー" icon={<Keyboard />}>
	ファイルを選択して **`F2`** キーを押すと、名前の変更が可能になります。
</Callout>

#### 📝 変更内容の記載例

ファイル名の末尾には、作業内容を具体的に記載してください。

| 作業内容             | ファイル名の記載例        |
| :------------------- | :------------------------ |
| **新商品を登録**     | `..._新商品追加.xlsm`     |
| **価格を変更**       | `..._価格改定.xlsm`       |
| **店舗の数量を変更** | `..._〇〇店数量変更.xlsm` |
| **終売処理**         | `..._終売処理.xlsm`       |
| **棚卸し等の修正**   | `..._在庫修正.xlsm`       |

</Step>
</Steps>
